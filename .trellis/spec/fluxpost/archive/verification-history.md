## PostgreSQL Diagnostics

Run read-only local PostgreSQL diagnostics from the project root:

```powershell
npm run db:diagnose
```

Useful filters:

```powershell
npm run db:diagnose -- --limit 10
npm run db:diagnose -- --run simple-...
npm run db:diagnose -- --scope openai/image --limit 20
```

This command uses `FLUXPOST_DIAG_DATABASE_URL` from the process or Windows user environment. Do not print or copy the secret URL. The diagnostic role is `fluxpost_reader`; it is read-only by default and can read app runtime tables, safe account/session views under `diagnostics`, and PostgreSQL stats/settings for lock/session diagnosis.
